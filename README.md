# wubby-tries-python

### Installation
1. Install Anaconda here: https://www.anaconda.com/download.
2. Install git: https://git-scm.com/.
3. Clone this repository.
```
git clone git@github.com:nikkopante/wubby-tries-python.git
```
4. Using your termina, go to the directory.
```
cd ~/wubby-tries-python
```
5. Run the following command to start Jupyter Notebook
```
jupyter notebook
```
